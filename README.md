# concept-aware-training
Code Release Status

This repository contains code for adapting LLaMA-style models to accept custom structured inputs and for training with a concept-aware objective function.

The code is currently being cleaned, documented, and prepared for public release following our preprint. We are actively finalizing documentation, tests, and example scripts to ensure the repository is easy to use and reproduce.

Access to the code at this time is available upon request. Please reach out to laya [at] stanford.edu.
